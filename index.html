Python

Python, yeni yazılıma başlayacaklar için iyi bir yoldaştır. Kolay bir yazılım dilidir.
Öğrenmesi sytax(sözdizimleri) diğer yazılım dillerine göre daha kolaydır. White space
kulllanır. Yani bir tab olarak ayarlar fonksiyonları, döngüleri, koşulları vb.
Örnek;
            def HelloWorld():
                print("Hello World")
Çıktı olarak: "Hello World" alırız.
eğer HelloWorld fonksiyonunu böyle yapsaydık hata alırdık.
                def HelloWorld():
                print("Hello World")
Çıktı olarak: 
            File "main.py", line 2
            print("Hello World")
            ^
        IndentationError: expected an indented block
Bu hata kısa olarak HelloWorld fonksiyonunda blok yok demektir.
